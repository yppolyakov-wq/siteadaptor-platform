# Демо-фото (CC0 / AI-набор)

Сюда кладутся РЕАЛЬНЫЕ фото для демо-сайтов (решение владельца 2026-07-10).
Пока файла для ключа нет — демо показывает тематический SVG-плейсхолдер (как раньше),
ничего не ломается. Полный список нужных ключей: `python manage.py demo_photo_report`.

## Правила

- **Лицензия:** только CC0/public domain ИЛИ сгенерированные AI. Никаких «бесплатных
  с атрибуцией» (CC-BY) — атрибуцию на витрине не показываем.
- **Имя файла** = ключевое слово кита, где всё не-буквенно-цифровое → дефис:
  `"bread,bakery"` → `bread-bakery.webp`. Форматы: `.webp` (лучше), `.jpg`, `.png`.
- **Фолбэк по токенам:** если точного файла нет, ищется по частям ключа —
  `bread.webp` покроет и `bread,rolls`, и `bread,loaf`. Так одним общим фото
  можно закрыть несколько позиций, а потом уточнять.
- **Варианты для галерей:** `<slug>-<N>.webp` (N = lock из отчёта) — разные фото
  для одинакового ключа в галерее.
- **Размеры:** товары/галереи ~800×600, hero ~1600×900, портреты ~400×400.
  Сжимайте (webp q≈80); цель <150 KB на файл.
- **После добавления фото** демо надо пересеять (URL пишется при сидинге):
  `python manage.py seed_demo_tenants --kit <kit> --recreate`.

Происхождение фиксируйте в `SOURCES.md` рядом (файл → откуда/лицензия) — чтобы
через год не гадать.
