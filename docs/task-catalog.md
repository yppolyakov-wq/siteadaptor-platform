1. **Новая работа получает ID отсюда** (строка в §3) ДО план-дока; план-док
   ссылается на этот ID.
2. **Расширяется → углубляется:** подзадачи получают точку (B1 → B1.1 →
   B1.1a), а НЕ новую букву/семейство.
3. **Новые семейства не плодим.** Если задача не ложится в существующие —
   добавляем её в §3 с ID `T-<n>` (сквозная нумерация) и пометкой источника.
4. При ссылке на коллидирующие коды уточнять семейство: «идея B1», «архетип
   A4», «эпик E-2» (не голое «B1»).
